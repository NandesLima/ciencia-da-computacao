## 🎯 Sobre este Repositório

Este repositório é uma curadoria para quem deseja cursar **Ciência da Computação** de forma autodidata e gratuita. O material é baseado no currículo da **OSSU** e adaptado para a plataforma **MkDocs**.

### 🛠️ Estrutura do Projeto
- [x] **MkDocs Integration:** Navegação por trilhas (Sistemas, Redes, Algoritmos, IA).
- [x] **Estética Neon:** Padronização visual com o portfólio **Ariel Shlomoh**.
- [x] **Continuous Deployment:** Deploy automático para GitHub Pages.

---

## 📚 Tópicos Principais

-   **Algoritmos & Estruturas de Dados:** Fundamentos da lógica.
-   **Sistemas de Computação:** Como o hardware e software se integram.
-   **Engenharia de Software:** Práticas de desenvolvimento profissional.
-   **Inteligência Artificial:** Machine Learning e processamento de dados.
-   **Redes & Segurança:** Infraestrutura e proteção de sistemas.
